### 🚀 Featured Projects

#### [PartsRadar – Indian PC Parts Price Tracker](https://partsradar.app)
A real-time PC parts price comparison platform for Indian retailers.

- Tracks thousands of PC component listings across vendors
- Shows price history, smart alerts, charts, and build guidance
- Built with **Python, Flask, Requests, Pandas, Plotly, Tailwind CSS**
- Project page: [PartsRadar case study](https://gaurav.vermabox.dev/project/partsradar)

#### [Beastly Neon YouTube Pipeline](https://gaurav.vermabox.dev/project/beastly-neon-youtube-pipeline)
An automated gaming-content pipeline that turns raw gameplay into YouTube-ready highlight reels.

- Detects, trims, and combines gameplay clips
- Adds transitions, music, end-screens, titles, descriptions, and tags
- Uploads and schedules videos using the **YouTube Data API**
- Tech: **Python, FFmpeg, SQL, API Automation**
- Channel: [@beastly_neon](https://www.youtube.com/@beastly_neon)

#### [Nether Teller / Whispers of the Nether Pipeline](https://gaurav.vermabox.dev/project/nether-teller-whispers-of-the-nether-pipeline)
A generative horror-story pipeline for short-form video creation.

- Generates horror stories using local LLMs
- Creates narration, AI visuals, subtitles, and edited scenes
- Uploads completed videos automatically to YouTube
- Tech: **Python, Local LLMs, TTS, AI Art, OpenCV, FFmpeg, YouTube API**
- Channel: [@NetherTeller](https://www.youtube.com/@NetherTeller)
